Remove state from the searcher chain, while keeping it in the executor.

This is used here to 'freeze' a state while it is waiting for other states in its merge group to reach the same instruction.

◆ mergeGroups

std::vector<MergeHandler *> klee::MergingSearcher::mergeGroups

Keeps track of all currently ongoing merges.

An ongoing merge is a set of states (stored in a MergeHandler object) which branched from a single state which ran into a klee_open_merge(), and not all states in the set have reached the corresponding klee_close_merge() yet.
